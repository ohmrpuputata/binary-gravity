# Security Policy

## Supported Versions

This project currently supports the active `1.21.1` Fabric line in the repository. Older snapshots or local experimental builds are not guaranteed to receive fixes.

## Reporting a Vulnerability

There is no private security intake configured for this repository yet. If you find a security issue, open a GitHub issue with enough detail to reproduce the problem, but avoid posting exploit automation, private tokens, or personal data.

For mod-specific issues, include:

- Minecraft version
- Fabric Loader version
- Fabric API version
- Java version
- Mod version or commit
- Whether the issue affects singleplayer, dedicated servers, or both

## Scope

Useful reports include crashes that can be triggered remotely on a server, world-corruption paths, unsafe file access, command abuse, privilege bypasses, or dependency vulnerabilities that affect shipped builds.
