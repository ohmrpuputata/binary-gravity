## Обзор

**Alien Apocalypse** — мод выживания для **Minecraft Fabric 1.21.1**, для одиночной и сетевой игры. Он превращает обычный мир в зону боевых действий с таймером: чем дольше вы живы, тем сильнее давит небо. Заражение расползается по земле, радиоактивные штормы превращают поверхность в логистическую головоломку, а рой перестаёт быть мешком с лутом и начинает вести себя как армия — обходит с флангов, роет туннели, бомбит, строит мосты, отступает и координирует атаки.

Всё связано одним правилом: **у каждого блока, моба и материала есть источник, применение и назначение.** Материалы ведут к снаряжению, снаряжение отвечает на конкретную угрозу, а лучшие награды спрятаны за самыми тяжёлыми боями — вплоть до **босса 8‑го дня**, с победой над которым вторжение заканчивается.

> **Названия.** Репозиторий на GitHub называется `binary-gravity`, проект Gradle — `alien-invasion`, а имя мода в игре — **Alien Apocalypse**.

Давление растёт по расписанию. Текущую фазу показывают HUD в игре и команда `/invasion`.

| День | Фаза | Что прилетает | Чем стоит заняться |
| :--: | :-- | :-- | :-- |

## Возможности

- <img src="docs/assets/icons/pulse.svg" width="18" align="absmiddle" alt=""> **Поэтапный темп вторжения** — мировые события, эскалация роя, заражение, радиация и кислотные дожди, привязанные к таймлайну по дням.
- <img src="docs/assets/icons/network.svg" width="18" align="absmiddle" alt=""> **Тактический ИИ пришельцев** — фланги, отступления, рытьё туннелей, бомбардировки, постройка мостов, агро отрядами, мародёрство, телепорт‑давление и поведение улья.
- <img src="docs/assets/icons/biohazard.svg" width="18" align="absmiddle" alt=""> **Заражение как прогрессия** — радиации нужна защита, заражению — очистка, технике — ЭМИ. Вперёд ведут контрмеры, а не голая сила.
- <img src="docs/assets/icons/shield.svg" width="18" align="absmiddle" alt=""> **Адаптивный рой** — с каждым днём рой учится пробивать обычную броню (ванильная всегда что‑то блокирует, но иммунитета не даёт). Инопланетная броня мода сбрасывает эту адаптацию — поэтому именно она и есть цель.
- <img src="docs/assets/icons/hive.svg" width="18" align="absmiddle" alt=""> **Контент мира** — города Роя, ульи, разбитые НЛО, лаборатории, бункеры, хранилища, жилы слизи и враждебный родной мир Роя.
- <img src="docs/assets/icons/blaster.svg" width="18" align="absmiddle" alt=""> **Арсенал со смыслом** — бластеры, гравитационное оружие, ЭМИ‑гранаты, плазменное и космическое снаряжение, нибириевые инструменты, палладиевая/платиновая наковальни и апекс‑оружие за боссом.
- <img src="docs/assets/icons/spark.svg" width="18" align="absmiddle" alt=""> **Клиентский лоск** — собственные модели и рендереры мобов, модели брони, HUD‑оверлей, частицы, эмбиент, полная локализация `en_us` + `ru_ru` и интеграция рецептов **JEI**.

## Угроза и ответ

Конструктивный хребет мода: **у каждой угрозы есть продуманная контрмера.** Сила оружия растёт вместе со сложностью препятствия, на которое оно отвечает, а не от одного дешёвого материала.

| Угроза | Ваш ответ |
| :-- | :-- |
| <img src="docs/assets/icons/radiation.svg" width="18" align="absmiddle" alt=""> Радиационные поля и штормы | Костюм химзащиты (полный сет = иммунитет), биофильтр‑маска, таблетки от радиации, портативный очиститель |
| <img src="docs/assets/icons/biohazard.svg" width="18" align="absmiddle" alt=""> Заражение и заражённая земля | Хитиновая броня (снимает заражение), космическая броня (иммунитет + ходьба по чужим блокам), биолопата |
| <img src="docs/assets/icons/swarm.svg" width="18" align="absmiddle" alt=""> Рой и толпы | Космический молот (слэм‑контроль), Звёздный тесак (рассекание), плазменное / иридиевое оружие |
| <img src="docs/assets/icons/shield.svg" width="18" align="absmiddle" alt=""> Адаптация к пробитию брони | Инопланетная броня мода (химзащита, хитин, космос) продолжает держать, когда ванильную уже «выучили» |
| <img src="docs/assets/icons/cog.svg" width="18" align="absmiddle" alt=""> Техника (буры, гравипушки) | ЭМИ‑граната |
| <img src="docs/assets/icons/gem.svg" width="18" align="absmiddle" alt=""> Апекс‑крафт | Глубокая радиоактивная добыча + ядра тёмной материи + дроп со структур и босса питают `bio_blade` — сильнейшее оружие |

## Установка

1. Установите Minecraft **1.21.1** с **Fabric Loader**.
2. Установите **Fabric API** для 1.21.1.
3. Положите собранный `alien-invasion-*.jar` в папку `mods`.
4. Запустите игру и найдите **Alien Apocalypse** в списке модов.

> Репозиторий не делает вид, что релиз существует до его публикации. Если GitHub‑релиза ещё нет — соберите мод из исходников (ниже).

---

## Сборка из исходников

**Windows**

```powershell
.\gradlew.bat build
```

**Linux / macOS**

```bash
chmod +x ./gradlew
./gradlew build
```

Перемапленный jar мода попадёт в `build/libs/`.

## Как помочь проекту

- Держите новый контент связанным с **добычей, применением и назначением** — материал без downstream‑использования это долг, а не глубина.
- Предпочитайте осмысленные контрмеры пауэр‑крипу: радиации нужна защита, рою — контроль толпы, технике — ЭМИ.
- Перед открытием pull request выполните `.\gradlew.bat build`.
- Обновляйте `en_us.json` и `ru_ru.json` вместе при добавлении любого видимого игроку текста.
- Полная картина — в [`CONTRIBUTING.md`](CONTRIBUTING.md), [`SECURITY.md`](SECURITY.md) и [`docs/DESIGN.md`](docs/DESIGN.md).
